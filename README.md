# Smart Attendance System with Face Recognition 🎯

A modern, full-stack attendance management system that uses **face recognition** to automatically mark student attendance. Built with Next.js, TypeScript, and face-api.js.

## ✨ Features

- ✅ **Face Recognition** - Mark attendance by just looking at the camera
- ✅ **User Authentication** - Secure login with NextAuth.js
- ✅ **Student Registration** - Register new students with face capture
- ✅ **Real-time Dashboard** - View attendance statistics and history
- ✅ **Analytics Dashboard** - Beautiful charts and insights
- ✅ **PDF Reports** - Download attendance reports
- ✅ **Mobile Responsive** - Works on all devices
- ✅ **Modern UI** - Glass-morphism design with animations

## 🛠️ Tech Stack

- **Frontend**: Next.js 14, TypeScript, TailwindCSS
- **Backend**: Next.js API Routes
- **Database**: SQLite with Prisma ORM
- **Face Recognition**: face-api.js (TensorFlow.js)
- **Authentication**: NextAuth.js
- **Charts**: Recharts
- **Animations**: Framer Motion
- **UI Components**: Heroicons, Headless UI

## 📋 Prerequisites

- Node.js 18+ 
- npm or yarn
- Webcam (for face capture)
